# Prompt Engineering Assignment

 Overview

This repository contains my Prompt Engineering assignment, which compares the performance of **zero-shot** and **few-shot** prompting across three AI models: **ChatGPT**, **Claude**, and **Gemini**.

The task was to classify 10 customer support messages into one of three categories:

* Complaint
* Question
* Praise

## Objectives

* Design a zero-shot prompt with no examples.
* Design a few-shot prompt using three labeled examples.
* Test both prompts on the same set of customer support messages.
* Compare the results and accuracy across different AI models.
* Analyze which prompting technique performs better.

## AI Tools Used

* ChatGPT
* Claude
* Gemini

## Key Takeaway

The comparison demonstrates how prompt design influences AI performance. While both zero-shot and few-shot prompting performed well on this classification task, few-shot prompting generally provides clearer guidance to the model and is expected to produce more consistent results on complex or ambiguous tasks.
